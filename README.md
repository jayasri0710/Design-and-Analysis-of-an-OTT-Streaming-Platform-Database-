# OTT Streaming Platform Database Project

## Project Overview
This project is a complete OTT Streaming Platform Database Management System developed using SQL.

The database simulates real-world OTT platforms like Netflix, Amazon Prime, and Disney+ Hotstar.

---

## Features
- User Management
- Multiple Profiles
- Movies and Series Management
- Episodes Tracking
- Watch History
- Continue Watching
- Ratings and Reviews
- Watchlist
- Subscription Plans
- Payments
- Device Management
- Active Sessions Tracking

---
